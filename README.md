当所有页面都添加完，可以点击“开始学习”进行半自动浏览__

__Sep5. 功能说明  
整个程序逻辑是事先保存所有要学习的页面，然后逐个游览，游览的时间为“学习时间”。  
由于20分会出现验证码，估默认是设置22分钟后自动游览列表的下一个页面。  
依次互逐个学习  
如果页面已经学完，倒计时还有很多时间，可以手动点击“跳过”，即可快速游览下一页__
