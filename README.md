## Info
__此软件为半自动化学习软件，学习过程的验证码还需手动输入，
但是游览的20分钟时间可以让软件帮助自动一页页进行游览。__

__整个程序逻辑是事先保存所有要学习的页面，然后逐个游览，游览的时间为“学习时间”。  
由于20分会出现验证码，故默认是设置22分钟后自动游览列表的下一个页面。  
依次逐个学习。  
如果页面已经学完，倒计时还有很多时间，可以手动点击“跳过”，即可快速游览下一页。  
当所有设定页面游览完成后，可以逐一打开需要验证码页面进行验证，随后可以继续一键学习。__
